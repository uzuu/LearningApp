# learningApp

learningApp нь шинээр сурсан зүйлээ бүртгэж, **forgetting curve** загвараар давтах өдрүүдийг календар дээр харуулдаг Expo (React Native) апп юм.

## Боломжууд

- Сурсан зүйлийн нэр + сурсан огноо бүртгэнэ.
- 1, 3, 7, 14, 30 хоногийн давталтын өдрүүдийг автоматаар тооцно.
- Давталтын өдрүүдийг календар дээр тэмдэглэж харуулна.
- Бүртгэлүүдээ утасны дотоод санах ойд хадгална (AsyncStorage).

## Ажиллуулах

```bash
npm install
npx expo start
```

Expo асаасны дараа:
- `a` Android
- `i` iOS
- `w` Web

## Төслийн бүтэц

- `App.js` — UI, forgetting curve тооцоолол, хадгалалт, календар.
- `package.json` — Expo болон dependency тохиргоо.
